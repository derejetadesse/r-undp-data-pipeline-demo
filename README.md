## 🎯 Project Overview

This project demonstrates a reproducible data processing and analysis pipeline built in R.  
It retrieves public economic indicators from the World Bank API, cleans and structures the data, and generates analytical outputs.

The workflow reflects real-world data analysis tasks used in policy research and economic development.

---

## ⚙️ Key Features

- Automated data retrieval using API (World Bank)
- Data cleaning and transformation using `dplyr`
- Visualization using `ggplot2`
- Reproducible workflow structure
- Organized project folders (scripts, data, outputs)

---

## 🌍 Use Case

This project simulates how analysts support organizations like UNDP by:

- Processing socio-economic data
- Generating insights for decision-making
- Building reusable data pipelines
